KeyTerm

Definition

ACI

Autonomic Computing Initiative

A framework to develop self-managing computing systems [1].

AEs

Autonomic Elements

Independent components that implement self-management functions [1].

AM

Autonomic Manager

A system component that manages resources automatically [1].

FHE

Fully Homomorphic Encryption

An encryption scheme that allows computation on encrypted data.

FaaS

Function As A Service

Provide a virtual environment for computer systems that can be dynamically scaled up and down.

IaaS

Infrastructure As A Service

Cloud-based virtualized computing resources such as servers and storage [1].

IoE

Internet Of Everything

A network connecting people, devices, and data for smarter interactions [1].
